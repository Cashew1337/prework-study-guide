HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The !DOCTYPE element is used to tell the browser what document type to expect(located in the head)

CSS

  • There are Inline, Internal, and External CSS sheets. It is best to work with External sheets whenever possible.
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • git status: Used to check what branch we are currently in.
  • git checkout: Used to move branches and can also be used to create new branches.
  • git pull: Used to pull code down to local workstation from selected repository.
  • git push: Used to push changes made to code to cloud repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the orfer in which a computer executes code in a script.
  • A function allows us to use arrays to call multiple items in a variable when the function is called.
